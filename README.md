# dearly note

A web app for reading and saving short notes.

### Features
- Typewriter intro: Forces a delay before the main content.
- Note wall: Displays a random note on a textured background.
- Save as image: Downloads the current note as a PNG.
- Categories: Different notes for different situations.

### Tech
- React + Tailwind
- Framer Motion (Animations)
- html2canvas (Image export)

### Setup
1. `npm install`
2. `npm run dev`
